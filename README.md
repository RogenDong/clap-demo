# clap-demo

## 预览交互
```
# help
Usage: cli [COMMAND]

Commands:
  啊啊啊   命令A 啊啊啊啊啊
  不不不   （说明）命令B
  quit  退出
  help  Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
```
```
# 啊啊啊 -h
命令A 啊啊啊啊啊

Usage: 啊啊啊 [OPTIONS] [NAME] [WORD]

Arguments:
  [NAME]  【name 就是 name】
  [WORD]  word 是什么 word?

Options:
  -l, --list  list 不是 list
  -h, --help  Print help
```
```
# 啊啊啊 aaa bbb
name: [aaa]
word: [bbb]
```
